🛒 Market Mar - E-commerce Local

¡Bienvenido al repositorio de Market Mar!
Este proyecto es la iniciativa de transformación digital de un negocio familiar tradicional de abastos (verduras, tubérculos y abarrotes) ubicado en el corazón del mercado de Cerro Azul, Perú.

El objetivo de esta plataforma es escalar las ventas del puesto físico hacia el entorno digital, permitiendo la venta directa a familias (B2C) y la captación de clientes mayoristas como restaurantes y hoteles locales (B2B).

🚀 Características Principales

Catálogo de Productos: Secciones divididas por Verduras, Tubérculos y Abarrotes.

Carrito de Compras Funcional: Uso de localStorage para que los productos seleccionados no se borren al navegar entre las diferentes páginas.

Integración con WhatsApp: Los pedidos del carrito generan automáticamente un mensaje detallado con el total a pagar y se envían directamente al WhatsApp del negocio (+51 910 535 496).

Sección B2B (Corporativo): Landing page específica para dueños de restaurantes que buscan precios al por mayor.

Totalmente Responsivo: Diseño adaptado para celulares, tablets y computadoras de escritorio.

🛠️ Tecnologías Utilizadas

HTML5: Estructura semántica de todas las páginas.

CSS3 & Bootstrap 5: Maquetación, sistema de grillas (Grid), tarjetas (Cards) y componentes responsivos.

Bootstrap Icons: Iconografía moderna para la interfaz.

Vanilla JavaScript: Lógica para el carrito de compras, cálculos de subtotales, notificaciones tipo "Toast" e interacción con la API de WhatsApp.

📂 Estructura del Proyecto

El proyecto consta de las siguientes vistas principales:

index.html: Página de inicio (Hero section y categorías destacadas).

nosotros.html: Historia familiar, Misión, Visión, Organigrama y análisis estratégico (FODA y PESTEL).

productos.html: Catálogo principal con filtros de categorías y botones de "Añadir al carrito".

ofertas.html: Promociones de la semana, combos y remates (ej. "Combo Cevichero").

carrito.html: Resumen de la compra con manipulación de cantidades y botón de checkout vía WhatsApp.

corporativo.html: Formulario de contacto y propuesta de valor para negocios/restaurantes.

login.html: Interfaz de inicio de sesión y registro de clientes.
